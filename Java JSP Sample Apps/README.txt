This project contains three Java Servlet / JSP apps that demonstrate the FindPerson, ReverseAddress, and ReversePhone API calls.

Built using Eclipse 4.5 ("Mars") on Windows 8, using Apache Tomcat 8.0 (Servlet API v 3.1, JSP v 2.3, EL v 3.0) and JSTL v 1.2.

Note there is a dependency on the proapi-client-1.0.2.jar library, provided by Whitepages, as well as the Jackson JSON API JAR libraries.
