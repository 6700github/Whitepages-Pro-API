This Android example shows a simple signup workflow using your phone number, and the ReversePhone Pro API endpoint.

It was built using Visual Studio 2013 on Windows, and Xamarin.Android 5.1.7.12.

There is a dependency on the Whitepages ProApiLibrary NuGet package, version 1.3.0, which in turn has a dependency on Newtonsoft's JSON.NET NuGet package.
